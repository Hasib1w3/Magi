Computer কী এবং এটা কীভাবে কাজ করে?

তুমি নিশ্চয়ই Computer দেখেছ — হয়তো বাসায়, স্কুলে, বা দোকানে। Computer হলো একটা বিদ্যুৎচালিত যন্ত্র (machine) যেটা তোমার দেওয়া নির্দেশ (instruction) মেনে কাজ করে। মোবাইল ফোনও একটা Computer — ছোট, কিন্তু কাজ একই।

🧠 সহজ উদাহরণ দিয়ে বুঝি

ধরো তুমি তোমার ছোট বোনকে বললে: "রান্নাঘর থেকে এক গ্লাস পানি নিয়ে আসো।" সে বুঝলো, গেল, এনে দিল। ঠিক একইভাবে, Computerকে তুমি নির্দেশ দাও — সে সেটা অনুসরণ করে। পার্থক্য হলো, Computer শুধু একটা নির্দিষ্ট ভাষায় বোঝে — সেই ভাষাকে বলে Programming Language। সেটা আমরা একটু পরে শিখব।

Computer-এর ভেতরে একটা অনেক গুরুত্বপূর্ণ অংশ থাকে যার নাম CPU (Central Processing Unit)। এটাকে বলে Computer-এর "মাথা" বা "ব্রেইন"। তুমি যা নির্দেশ দাও, CPU সেটা হিসাব করে এবং কাজ করে।

💡 মনে রাখো
Computer-কে নির্দেশ দেওয়ার জন্য যে বিশেষ ভাষা ব্যবহার করা হয় সেটার নাম Programming Language। আর যে মানুষ এই ভাষায় নির্দেশ লেখে তাকে বলে Programmer বা Developer। তুমিও এই Class-গুলো শেষে একজন Developer হয়ে যাবে! 🎯

Internet এবং Web Browser কী?

Internet হলো পৃথিবীজুড়ে লক্ষ কোটি Computer-এর একটা বিশাল নেটওয়ার্ক (network) — মানে সব Computer একে অপরের সাথে সংযুক্ত (connected)। তুমি যখন YouTube দেখো বা Google-এ কিছু সার্চ করো, তখন তোমার ফোন বা Computer এই Internet-এর মাধ্যমে অনেক দূরের আরেকটা Computer (যাকে বলে Server) থেকে তথ্য নিয়ে আসে।

🧠 সহজ উদাহরণ

ধরো Internet হলো একটা বিশাল রাস্তার নেটওয়ার্ক, যেটা দিয়ে সারা দুনিয়ায় তথ্য (information/data) যাতায়াত করে। আর তোমার ফোন বা Computer হলো একটা বাড়ি যেটা ওই রাস্তার সাথে যুক্ত।

এখন আসি Web Browser-এ। Web Browser হলো একটা Software (program) যেটা দিয়ে তুমি Internet-এ ঘুরতে পারো এবং Website দেখতে পারো।

উদাহরণ ১ বিখ্যাত Web Browser-গুলো
তুমি নিশ্চয়ই এগুলো দেখেছ বা ব্যবহার করেছ:
  • 🔵 Google Chrome — সবচেয়ে বেশি ব্যবহৃত Browser
  • 🟠 Mozilla Firefox — আরেকটা জনপ্রিয় Browser
  • 🔷 Microsoft Edge — Windows Computer-এ থাকে
  • 🍎 Safari — iPhone ও Mac-এ থাকে
তোমার ফোনে এখন যেটা দিয়ে Website দেখো — সেটাই তোমার Web Browser।
উদাহরণ ২ Browser কীভাবে কাজ করে — সহজ ভাষায়
তুমি Google Chrome-এ গিয়ে টাইপ করলে: www.youtube.com

এই ঘটনাগুলো ঘটে:
  • ১. Chrome তোমার Internet-এর মাধ্যমে YouTube-এর Server-এ একটা request পাঠায় — মানে বলে "ভাই, YouTube-এর পাতাটা দাও!"
  • ২. YouTube-এর Server response দেয় — মানে পাতার সব তথ্য পাঠিয়ে দেয়।
  • ৩. Chrome সেই তথ্য নিয়ে তোমার স্ক্রিনে সুন্দর করে সাজিয়ে দেখায়।
এই পুরো কাজটা এত দ্রুত হয় যে তুমি টের পাওই না!

Website কী এবং এটা কোথায় থাকে?

Website হলো Internet-এ থাকা একটা পাতা বা পাতার সমষ্টি যেটা তুমি Web Browser-এ দেখতে পারো। যেমন: YouTube, Facebook, Wikipedia — এগুলো সবই Website।

এই Website-গুলো কোথায় থাকে? সেগুলো থাকে বিশেষ শক্তিশালী Computer-এ — যার নাম Server। Server মানে এমন একটা Computer যেটা সবসময় চালু থাকে এবং যেকোনো মানুষ Internet-এর মাধ্যমে সেখান থেকে তথ্য নিতে পারে।

🧠 সহজ উদাহরণ

ধরো একটা বিশাল Library (পাঠাগার) আছে — সেটাই Server। সেই Library-তে অনেক বই আছে — সেগুলোই Website। আর তুমি Library-তে যে রাস্তা দিয়ে যাও সেটা Internet। তুমি যখন Library-তে গিয়ে বই পড়ো — সেটাই Web Browser-এ Website দেখা।

প্রতিটা Website-এর একটা নিজস্ব ঠিকানা আছে — যার নাম URL (Uniform Resource Locator)। যেমন: www.google.com বা www.facebook.com — এগুলো হলো URL। এটাকে ভাবো বাড়ির ঠিকানার মতো — প্রতিটা বাড়ির আলাদা ঠিকানা থাকে, তেমনি প্রতিটা Website-এর আলাদা URL থাকে।

💡 একটা Website আসলে কী দিয়ে তৈরি?

প্রতিটা Website মূলত তিনটা জিনিস দিয়ে তৈরি:

  • 🏗️ HTML (HyperText Markup Language) — এটা Website-এর কাঠামো তৈরি করে। যেমন বাড়ির ইট-পাথর।
  • 🎨 CSS (Cascading Style Sheets) — এটা Website-কে সুন্দর করে সাজায়। যেমন বাড়ির রঙ ও ডিজাইন।
  • ⚡ JavaScript — এটা Website-কে জীবন্ত ও কার্যকর করে। যেমন বাড়ির বিদ্যুৎ — আলো জ্বলে, ফ্যান চলে।
আমরা JavaScript শিখছি — এটাই সবচেয়ে মজার অংশ!

Programming Language কী?

তুমি আর আমি কথা বলি বাংলায় বা ইংরেজিতে — এগুলো Human Language (মানুষের ভাষা)। কিন্তু Computer মানুষের ভাষা বোঝে না। Computer আসলে শুধু বোঝে 0 আর 1 — যেটাকে বলে Binary।

🧠 সহজ উদাহরণ

মনে করো তুমি একজন বাংলাদেশি, আর তোমার বন্ধু শুধু জাপানি ভাষা বোঝে। তোমাদের মাঝে কথা বলতে হলে একজন Translator (অনুবাদক) দরকার। Programming Language হলো সেই Translator — তুমি Code লেখো, আর সেই Code Computer-এর ভাষায় (0 ও 1-এ) রূপান্তরিত হয়ে Computer-কে বোঝানো হয়।

পৃথিবীতে অনেক রকমের Programming Language আছে। কিছু বিখ্যাত নাম:

উদাহরণ ১ বিখ্যাত Programming Language-গুলো
  • ⚡ JavaScript — Website-কে জীবন্ত করে (আমরা এটাই শিখব!)
  • 🐍 Python — AI, Data Science-এ ব্যবহার হয়
  • ☕ Java — Android app তৈরিতে ব্যবহার হয়
  • 🎮 C++ — গেম তৈরিতে ব্যবহার হয়
এগুলো আলাদা আলাদা ভাষা, আলাদা আলাদা কাজে লাগে। কিন্তু সবই শেখা যায়!
উদাহরণ ২ Code আসলে কী দেখতে?

এটা একটা JavaScript Code-এর উদাহরণ। ভয় পেও না — এখন না বুঝলেও চলবে, পরে সব বুঝবে:

JavaScript 
// এটা একটা সহজ JavaScript code
let name = "রাফি";
console.log("হ্যালো, " + name);
Output (ফলাফল)
হ্যালো, রাফি
এই Code চালালে Computer লিখে দেবে: হ্যালো, রাফি — এটাই Programming!

JavaScript কী এবং কেন শিখব?

JavaScript (JS নামেও পরিচিত) হলো একটি Programming Language। এটা ১৯৯৫ সালে তৈরি হয়েছিল, এবং এখন এটা পৃথিবীর সবচেয়ে বেশি ব্যবহৃত Programming Languageগুলোর একটা।

JavaScript-এর বিশেষত্ব হলো — এটা সরাসরি Web Browser-এ চলে। মানে তুমি Chrome বা Firefox খুললেই সেখানে JavaScript চালাতে পারবে — আলাদা কোনো বড় Software লাগবে না!

⚡ গুরুত্বপূর্ণ তথ্য — নাম নিয়ে বিভ্রান্তি!

JavaScript এবং Java — নাম দেখে মনে হয় একই জিনিস, কিন্তু আসলে এরা সম্পূর্ণ আলাদা! এটা যেন "ইন্দোনেশিয়া" আর "ইন্ডিয়া" — নামে মিল আছে, কিন্তু সম্পূর্ণ ভিন্ন দেশ। তাই কখনো গুলিয়ে ফেলবে না।

উদাহরণ ১ JavaScript দিয়ে কী কী করা যায়?
  • 🎮 Browser-এ গেম বানানো — যেমন Flappy Bird টাইপের গেম!
  • 🛒 E-commerce Website — যেখানে জিনিস কিনতে পারবে, Cart-এ যোগ করতে পারবে
  • 💬 Chat App — WhatsApp-এর মতো messaging app
  • 📊 Dashboard — সুন্দর গ্রাফ ও চার্ট দেখানো
  • 🤖 এমনকি AI-ও — ChatGPT-এর Website-ও JavaScript দিয়ে তৈরি!
উদাহরণ ২ JavaScript ছাড়া Website কেমন হতো?

মনে করো তুমি Facebook-এ Like বাটনে ক্লিক করলে — সাথে সাথে সংখ্যাটা বাড়লো। এটা JavaScript-এর কাজ।

JavaScript না থাকলে — পাতা reload (মানে পুরো পাতা আবার লোড) করতে হতো শুধু একটা Like দিতে। JavaScript Website-কে dynamic (গতিশীল, সজীব) করে তোলে। JavaScript ছাড়া Website হবে একটা স্থির ছবির মতো — ক্লিক করলে কিছুই হবে না।

আমাদের প্রথম JavaScript Code লেখা!

এবার মজার অংশ! তুমি এখনই নিজের Computer বা ফোনে JavaScript চালাবে। কোনো বড় Software লাগবে না — শুধু Web Browser লাগবে।

🔧 Console কী?

Web Browser-এর ভেতরে একটা গোপন টুল আছে যার নাম Console। এটা Developer-দের জন্য তৈরি একটা জায়গা যেখানে তুমি সরাসরি JavaScript Code লিখে চালাতে পারো। Console খুলতে হয় F12 চেপে, তারপর "Console" ট্যাবে ক্লিক করে। (ফোনে এটা একটু কঠিন, Computer-এ বেশি সহজ।)

এখন নিচের ধাপগুলো অনুসরণ করো:

হাতে-কলমে Console খুলে প্রথম Code চালাও
  • ১. Google Chrome Browser খোলো।
  • ২. Keyboard-এ F12 চাপো (অথবা Mouse-এর Right Click করে "Inspect" বা "পরীক্ষা করো" তে ক্লিক করো)।
  • ৩. ওপরে অনেকগুলো ট্যাব দেখবে — সেখান থেকে "Console" তে ক্লিক করো।
  • ৪. নিচে একটা খালি জায়গা দেখবে যেখানে টাইপ করা যায়।
  • ৫. সেখানে এটা টাইপ করো এবং Enter চাপো:
Console-এ টাইপ করো 
console.log("আমি JavaScript শিখছি!");
তুমি দেখতে পাবে
আমি JavaScript শিখছি!

অভিনন্দন! 🎉 তুমি এইমাত্র তোমার জীবনের প্রথম JavaScript Code চালালে!

এখন এই Code-টা একটু ভাঙি বুঝিয়ে:

🔍 console.log() বিস্তারিত
  • console — এটা Browser-এর একটা অংশের নাম। এটাকে ভাবো একটা নোটবুকের মতো।
  • .log() — এটা একটা Function (কাজ করার নির্দেশ)। log মানে "লিখে রাখো"। তুমি যা লিখবে সেটা Console-এ দেখাবে।
  • ভেতরের "..." — দুটো উদ্ধৃতি চিহ্নের ভেতরে যা লিখবে সেটাই দেখাবে।
  • ; (semicolon) — প্রতিটা নির্দেশের শেষে এটা দেওয়া হয়। বাংলায় যেভাবে বাক্যের শেষে দাঁড়ি (।) দাও, এটা তেমনই।
উদাহরণ ১ নিজের নাম দিয়ে চেষ্টা করো
এবার Console-এ এটা টাইপ করো (তোমার নাম দিয়ে):
JavaScript 
console.log("আমার নাম রাফি।");
console.log("আমি JS শিখছি!");
Output
আমার নাম রাফি।
আমি JS শিখছি!
দেখলে? দুটো আলাদা console.log() দিলে দুটো আলাদা লাইনে দেখায়!

✅ Class 1 Summary
  • Computer হলো একটা যন্ত্র যেটা নির্দেশ মেনে কাজ করে।
