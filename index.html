🖥️ শুরুর আগে: Foundation

একদম গোড়া থেকে শুরু — কম্পিউটার থেকে Python পর্যন্ত

💻 কম্পিউটার আসলে কী করে?

কল্পনা কর তোমার একটা চাকর আছে — কিন্তু সে অনেক বোকা। সে শুধু যা বলবে তাই করবে, একটুও নিজে বুদ্ধি খাটাবে না। কিন্তু সে অবিশ্বাস্য রকম দ্রুত কাজ করে — প্রতি সেকেন্ডে কোটি কোটি কাজ করতে পারে। এই চাকরটার নামই হলো কম্পিউটার।

কম্পিউটারের তিনটা মূল অংশ বুঝতে হবে। এটা বুঝলে পরে Python কেন কিছু কিছু কাজ করে তা নিজেই বুঝতে পারবি।

🧠 CPU (Central Processing Unit) — মস্তিষ্ক

CPU মানে হলো কম্পিউটারের মস্তিষ্ক। সব হিসাব-নিকাশ এখানেই হয়। তুমি যখন বলবে "১ + ১ কত?" — CPU সেটা বলবে "২"। CPU প্রতি সেকেন্ডে কোটি কোটি ছোট ছোট calculation করতে পারে।

📝 RAM (Random Access Memory) — কাজের টেবিল

RAM হলো কম্পিউটারের কাজের টেবিল। যখন তুমি কোনো program চালাও, সেটার সব data RAM-এ আসে। কিন্তু মনে রাখবি — RAM অনেক দ্রুত কিন্তু বিদ্যুৎ চলে গেলে সব মুছে যায়। তুমি যদি কোনো Python program চালাও এবং variable তৈরি করো, সেটা RAM-এ থাকে।

💾 Storage (HDD/SSD) — আলমিরা

Storage হলো কম্পিউটারের আলমিরা। তোমার সব ফাইল, ছবি, Python code — সব এখানে থাকে। বিদ্যুৎ চলে গেলেও এখান থেকে কিছু মুছে যায় না। কিন্তু RAM-এর তুলনায় এটা অনেক ধীর।

ধর তুমি গণিত করছ। তোমার বই (Storage) থেকে অঙ্কটা পড়লে, খাতায় (RAM) লিখলে এবং মাথায় (CPU) হিসাব করলে। অঙ্ক শেষে খাতার হিসাব মুছে যায়, কিন্তু বই ঠিকই থাকে।

📋 Program কী? Programming Language কী?

মনে কর তুমি তোমার বোকা চাকরকে (কম্পিউটার) বাজারে পাঠাবে। তাকে বললে "বাজারে যা" — সে চলে যাবে কিন্তু কী কিনবে জানবে না। তোমাকে পুরো list দিতে হবে: "প্রথমে দোকানে যাও, তারপর ১ কেজি চাল নাও, তারপর ২টা পেঁয়াজ নাও, তারপর টাকা দাও, তারপর বাড়িতে ফিরে আসো।" এই step-by-step নির্দেশনার নামই হলো Program।

একটা Program হলো কম্পিউটারকে দেওয়া step-by-step নির্দেশনা। Program ছাড়া কম্পিউটার কিছুই করতে পারে না — এমনকি screen-এ কিছু দেখাতেও পারে না।

কম্পিউটার কি সরাসরি বাংলা বোঝে?

না! কম্পিউটার শুধু বোঝে ০ আর ১ — এটাকে বলে Binary বা machine language। একটা সাধারণ program লিখতে গেলে লক্ষ লক্ষ ০ এবং ১ লিখতে হতো। এটা মানুষের পক্ষে লেখা অসম্ভব।

তাই মানুষ Programming Language তৈরি করেছে — যেটা মানুষের ভাষার কাছাকাছি। তুমি ইংরেজিতে লিখবে print("Hello") এবং কম্পিউটার সেটাকে নিজেই ০ এবং ১-এ রূপান্তর করে নেবে।

💡 বড় ভাইয়ের টিপস: Programming Language হলো মানুষ এবং কম্পিউটারের মাঝে একটা translator (অনুবাদক)। তুমি human-friendly ভাষায় লেখো, সে computer-friendly ভাষায় translate করে।

🐍 Python কী? কেন শিখবে?

Programming language-গুলো বিভিন্ন রকম হয়। কেউ অনেক কঠিন কিন্তু অনেক শক্তিশালী (যেমন C++ — রেসের গাড়ির মতো, দ্রুত কিন্তু চালানো কঠিন)। কেউ সহজ এবং বহুমুখী। Python হলো সেই ভাষা যেটা সহজ কিন্তু অনেক শক্তিশালী — যেন একটা আরামদায়ক গাড়ি যেটা যেকোনো রাস্তায় চলে।

⚙️ Python Install করা — Step by Step

Install করা মানে হলো Python-এর program তোমার computer-এ নামিয়ে সেট করা, যাতে তুমি Python code চালাতে পারো।

Windows-এ Install:

  1. Browser-এ যাও: python.org — এটা Python-এর official website
  2. "Downloads" বাটনে click করো — সর্বশেষ version download হবে
  3. Downloaded file (.exe) খোলো। গুরুত্বপূর্ণ: "Add Python to PATH" checkbox-এ tick দাও — এটা না দিলে পরে সমস্যা হবে
  4. "Install Now" click করো এবং অপেক্ষা করো
  5. Installation শেষে "Close" দাও

Mac-এ Install:

  1. Browser-এ যাও: python.org
  2. "Downloads" → "macOS" থেকে latest version নামাও
  3. .pkg file খুলে install করো

VS Code — Code Editor কী?

Code Editor হলো সেই program যেখানে তুমি Python code লিখবে। এটা Notepad-এর মতো কিন্তু অনেক বেশি smart — code রঙিন করে দেখায়, ভুল ধরে দেয়, suggest করে।

VS Code (Visual Studio Code) — এটা Microsoft-এর তৈরি, বিনামূল্যে, এবং পুরো পৃথিবীতে সবচেয়ে জনপ্রিয় code editor। code.visualstudio.com থেকে download করো।

VS Code install করার পরে Python Extension install করো — VS Code-এর মধ্যেই Extensions বাটনে গিয়ে "Python" লিখে search করো এবং Microsoft-এর টা install করো।

File এবং .py Extension

File extension হলো file-এর নামের শেষে যে অংশ — এটা বলে দেয় file-টা কী ধরনের। যেমন: .jpg মানে ছবির file, .mp3 মানে গানের file।

Python code-এর file-এ .py extension থাকে। যেমন: hello.py, calculator.py। Python দেখলেই বোঝে এটা Python-এর code।

💻 Terminal কী? Python কীভাবে চালাবে?

তুমি সারাদিন smartphone-এ apps ব্যবহার করো — সব কিছু button চেপে চেপে। কিন্তু professional programmer-রা অনেক কাজ text লিখে করে — যেমন পুরনো দিনের computer-এ সব কিছু command দিয়ে করতে হতো। এই text দিয়ে computer-কে command দেওয়ার জায়গাটার নামই Terminal বা Command Prompt।

Terminal (Windows-এ Command Prompt বা PowerShell) হলো একটা জানালা যেখানে তুমি text command লিখে computer-কে নির্দেশ দিতে পার।

Terminal কীভাবে খুলবে:

Windows: Start button → "cmd" লিখে search → Enter দাও

Mac: Spotlight (Cmd+Space) → "Terminal" লিখে Enter

Python file চালানো — প্রতিটা অক্ষর বুঝি:

Terminal 
python hello.py

এই command-এর প্রতিটা অংশ বোঝা যাক:

  • python — এই শব্দটা computer-কে বলছে: "Python interpreter ব্যবহার করো।" Interpreter মানে হলো সেই program যে তোমার Python code পড়ে এবং computer-কে সেটা বোঝায়।
  • স্পেস — python এবং file-এর নামকে আলাদা করছে
  • hello.py — এটা file-এর নাম। Python interpreter এই file খুলবে এবং ভেতরের code পড়বে।

Enter চাপলে ভেতরে কী ঘটে?

  1. Computer python command দেখে Python interpreter চালু করে
  2. Interpreter hello.py file খোলে এবং প্রথম লাইন থেকে পড়া শুরু করে
  3. প্রতিটা লাইন উপর থেকে নিচে একে একে পড়ে এবং execute (কার্যকর) করে
  4. শেষ লাইন শেষ হলে program বন্ধ হয়
Python কেন উপর থেকে নিচে পড়ে? এটাই natural। যেমন তুমি বই পড়ো — প্রথম পাতা থেকে শুরু করো। Python-ও তাই করে। এই concept-টা সব জায়গায় কাজে লাগবে।
এখন বুঝলি computer কীভাবে কাজ করে এবং Python কীভাবে চালাতে হয়। এবার সত্যিকারের মজা শুরু হচ্ছে — তুমি Python-কে প্রথমবার কথা বলাবে! পরের অধ্যায়ে শিখবি print() — Python-এর সবচেয়ে প্রথম এবং সবচেয়ে গুরুত্বপূর্ণ command।

✍️ প্রথম কোড — Python-কে কথা বলানো

তোমার জীবনের প্রথম Python code লেখার সময় এসেছে!

🖨️ print() — Python-কে কথা বলানো

কল্পনা কর তোমার কাছে একটা magic কাগজ আছে। তুমি যা লিখবে, সেটা screen-এ দেখাবে। কিন্তু এই magic কাগজকে activate করতে একটা spell লাগে — সেই spell-এর নাম print()। যেকোনো কিছু screen-এ দেখাতে চাইলে এই spell বলতে হবে।

Function কী? (Preview)

Function হলো একটা কাজের বান্ডেল যেটাকে একটা নাম দেওয়া হয়েছে। print() হলো এরকমই একটা function — Python-এর দল এটা আগে থেকেই তৈরি করে রেখেছে। এই ধরনের আগে থেকে তৈরি function-কে বলে built-in function। Function সম্পর্কে পরে অনেক বিস্তারিত শিখবি — এখন শুধু ব্যবহার করতে শেখো।

print নামটা কেন?

পুরনো দিনে "print" মানে ছিল printer-এ কিছু ছাপানো। এখন এর মানে screen-এ কিছু দেখানো। নামটা ঐতিহাসিক কারণে এখনও print-ই আছে।

() কেন লাগে?

() মানে হলো "এই function-কে এখন কাজে লাগাও।" এটা function call করার চিহ্ন। () ছাড়া লিখলে Python শুধু জানবে print একটা কিছু আছে, কিন্তু কাজ করাবে না।

String কী? "" কেন লাগে?

String হলো যেকোনো text — অক্ষর, শব্দ, বাক্য। Computer-কে বলতে হয় "এটা text, সংখ্যা নয়।" এই কাজটা করে quotation marks (" " অথবা ' ')।

Data type মানে হলো data কী ধরনের — সংখ্যা? text? হ্যাঁ/না? এটা পরে বিস্তারিত বুঝবি।

Python — hello.py 
print("আমার নাম রাহিম")
print("আমি Python শিখছি")
print(42)           # সংখ্যায় quotes লাগে না
print("42")        # এটা text হিসেবে 42
print(3.14)        # দশমিক সংখ্যাও চলে
print(42) vs print("42") পার্থক্য কী?
print(42) — এখানে 42 একটা সংখ্যা। Python এটা দিয়ে math করতে পারবে।
print("42") — এখানে "42" একটা text। Python এটাকে "চার" "দুই" অক্ষর হিসেবে দেখে, সংখ্যা হিসেবে নয়। এই পার্থক্যটা পরে অনেক গুরুত্বপূর্ণ হবে।

📦 Variable — তথ্যের বাক্স

কল্পনা কর তোমার ঘরে অনেক বাক্স আছে। প্রতিটা বাক্সের উপর একটা লেবেল লাগানো — "বই", "জামা", "খেলনা"। তুমি যখন বই দরকার হবে, "বই" লেবেলের বাক্স খুলবে। Computer-এও ঠিক এরকম বাক্স থাকে RAM-এ — এই বাক্সগুলোকে বলে Variable।

Variable হলো RAM-এ একটা নামের জায়গা যেখানে data রাখা যায়। পরে সেই নাম ব্যবহার করে data বের করা যায়।

Variable তৈরি করা:

🏷️ Data Types — তথ্য কত ধরনের

দুনিয়ায় বিভিন্ন ধরনের তথ্য আছে। তোমার নাম text, বয়স সংখ্যা, উচ্চতা দশমিক সংখ্যা, "পরীক্ষায় পাস করেছ কিনা" এটা হ্যাঁ বা না। Computer-ও এই পার্থক্য বোঝে — এবং ভিন্ন ধরনের data ভিন্নভাবে handle করে।

int — পূর্ণ সংখ্যা

int (Integer) মানে পূর্ণ সংখ্যা — ভাগের চিহ্ন নেই। যেমন: ১, ২, ১৫, -৩, ১০০০।

float — দশমিক সংখ্যা

float মানে দশমিক সংখ্যা। যেমন: ৩.১৪, ৫.৬, -২.৫। int থেকে আলাদা কারণ computer-এ দশমিক সংখ্যা ভিন্নভাবে store হয়।

str — String (Text)

str (String) মানে যেকোনো text। Single quote ('hello') বা double quote ("hello") দুটোই চলে — তুমি যেটা দিয়ে শুরু করবে সেটা দিয়েই শেষ করতে হবে।

bool — Boolean (হ্যাঁ বা না)

bool মানে শুধু দুটো মান: True বা False। কেন শুধু দুটো? কারণ computer ভেতরে শুধু ০ আর ১ বোঝে — False = ০, True = ১। এটা condition check করতে (পরে শিখবি) অনেক দরকার।

None — কিছু নেই

None মানে "কোনো মান নেই।" এটা 0 বা "" থেকে আলাদা — 0 মানে শূন্য আছে, "" মানে খালি text আছে, কিন্তু None মানে একেবারেই কিছু নেই।

🔄 Type Conversion — এক type থেকে অন্য type

"৪২" এবং ৪২ — দুটো দেখতে একই কিন্তু computer-এর কাছে সম্পূর্ণ আলাদা। "৪২" হলো text (এর সাথে math করা যায় না), ৪২ হলো সংখ্যা (এর সাথে math করা যায়)। কখনো কখনো এক type-কে অন্য type-এ রূপান্তর করতে হয়।
Python — conversion.py 
# String থেকে int:

# int থেকে string:
age = 15

print("বয়স: " + age_str)   # বয়স: 15

# String থেকে float:
pi = float("3.14")        # "3.14" → 3.14

# এটা error দেবে:
# int("hello")  # "hello" কে সংখ্যায় রূপান্তর করা অসম্ভব!
int("hello") কেন error দেবে? "hello" একটা text যেটাকে কোনোভাবেই সংখ্যায় রূপান্তর করা সম্ভব নয়। Python তখন ValueError দেয় — মানে "তুমি যা চাইলে সেটা সম্ভব না।" Error handling পরে বিস্তারিত শিখবি।

🛠️ Function তৈরি ও Call করা

তুমি যখন রান্না করো, প্রতিবার scratch থেকে শুরু করো না — রেসিপি আছে। "ভাত রান্না রেসিপি" একবার লেখো, তারপর যতবার দরকার ততবার সেই রেসিপি follow করো। Programming-এ function ঠিক এরকম রেসিপি — একবার লেখো, বারবার ব্যবহার করো।

DRY Principle — "Don't Repeat Yourself" — একই code বারবার লেখা খারাপ। Function দিয়ে একবার লিখলে বারবার call করা যায়।

Python — functions.py 
# Function define করা (রেসিপি লেখা):
def greet():
    print("হ্যালো! কেমন আছ?")
    print("Python শিখছ ভালো করছ!")

# Function call করা (রেসিপি use করা):
greet()     # এখন চলবে
greet()     # আবার চলবে
greet()     # যতবার চাই

# Parameters সহ function:
def greet_person(name):        # name হলো parameter
    print(f"হ্যালো, {name}!")

greet_person("রাহিম")    # "রাহিম" হলো argument
greet_person("করিম")    # এবার করিম
Parameter vs Argument পার্থক্য:
Parameter: function define করার সময় যে নাম লেখো — সেটা placeholder
Argument: function call করার সময় যে actual value দাও
"রেসিপিতে লেখা 'সবজি'" = parameter। তুমি রান্নায় "পটল" ব্যবহার করলে = argument।

↩️ Parameters, Return এবং তাদের পার্থক্য

Python — return.py 
# Default parameter:
def greet(name="বন্ধু"):         # name না দিলে "বন্ধু" use হবে
    print(f"হ্যালো, {name}!")

greet()              # হ্যালো, বন্ধু!
greet("রাহিম")      # হ্যালো, রাহিম!

def add(a, b):
    result = a + b
    return result    # এই মানটা caller-কে ফেরত দাও

print(total)

# Multiple values return:
def min_max(numbers):
    return min(numbers), max(numbers)  # tuple return

print(lo, hi)        # 1 7

# *args — যেকোনো সংখ্যক argument:
def sum_all(*args):  # args হবে একটা tuple
    return sum(args)

print(sum_all(1, 2, 3))       # 6
print(sum_all(10, 20, 30, 40)) # 100

# **kwargs — named arguments:
def student_info(**kwargs):  # kwargs হবে একটা dict
    for key, val in kwargs.items():
        print(f"{key}: {val}")

student_info(name="রাহিম", age=15, grade="নবম")
⚠️ Return vs Print — এই পার্থক্য অনেক গুরুত্বপূর্ণ!
print() → screen-এ দেখায়, কিন্তু program কিছু পায় না।
return → function-এর caller-কে একটা মান ফেরত দেয়।

উদাহরণ: x = add(5, 3) — যদি add() print করে কিন্তু return না করে, তাহলে x = None হবে!
Real code-এ প্রায় সবসময় return ব্যবহার হয়।

Absolute path: পুরো ঠিকানা — যেমন C:\Users\Rahim\notes.txt

Relative path: এখন যেখানে আছি সেখান থেকে — যেমন notes.txt মানে একই folder-এ।

    school_name = "বাংলা হাই স্কুল"

✨ Magic Methods ও Property

Python-এ কিছু special method আছে যাদের নাম __ দিয়ে শুরু এবং __ দিয়ে শেষ। এগুলোকে dunder (double underscore) বা magic method বলে। এগুলো Python নিজে call করে — তুমি করো না। Real production code-এ সবখানে এগুলো দেখবে।
Python — magic_methods.py 
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self._balance = balance  # _ মানে "private, বাইরে সরাসরি access করো না"

    def __str__(self):
        return f"Account({self.owner}: ৳{self._balance})"

    def __repr__(self):
        return f"BankAccount(owner='{self.owner}', balance={self._balance})"

    def __len__(self):
        return int(self._balance)    # len(account) call করলে

    def __add__(self, amount):      # account + 500
        return BankAccount(self.owner, self._balance + amount)

    # @property — getter:
    @property
    def balance(self):
        return self._balance

    # @balance.setter — setter:
    @balance.setter
    def balance(self, amount):
        if amount < 0:
            raise ValueError("Balance ঋণাত্মক হতে পারে না!")
        self._balance = amount

    @classmethod
    def from_dict(cls, data):        # classmethod — cls = class নিজে
        return cls(data["owner"], data["balance"])

    @staticmethod
    def is_valid_amount(amount):    # staticmethod — self বা cls নেই
        return amount > 0

Generator — মেমোরি বাঁচাও

# yield দিয়ে generator তৈরি:
def count_up(n):
    i = 0
    while i < n:
        yield i      # return নয়, yield — pause করে value দেয়
        i += 1

for num in count_up(5):
    print(num)        # 0 1 2 3 4 — একটা একটা করে আসে

# Generator expression (list comprehension-এর মতো কিন্তু ()):

# Decorator তৈরি:
def timer(func):
    import time
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)  # আসল function call
        end = time.time()
        print(f"{func.__name__} চললো {end-start:.4f} সেকেন্ড")
        return result
    return wrapper

# @ দিয়ে apply করো:
@timer
def slow_function():
    import time
    time.sleep(1)
    print("কাজ শেষ!")

from functools import reduce

numbers = [1, 2, 3, 4, 5]

# map() — প্রতিটা element-এ function apply করো:
doubled = list(map(lambda x: x * 2, numbers))
print(doubled)    # [2, 4, 6, 8, 10]

# filter() — condition দিয়ে ছেঁকে নাও:
evens = list(filter(lambda x: x % 2 == 0, numbers))
print(evens)      # [2, 4]

# reduce() — সব combine করো:
total = reduce(lambda acc, x: acc + x, numbers)
print(total)      # 15 (1+2+3+4+5)

# zip() — দুটো list একসাথে:
names = ["রাহিম", "করিম", "সালমা"]
scores = [85, 90, 78]
for name, score in zip(names, scores):
    print(f"{name}: {score}")

# enumerate() — index এবং value একসাথে:
for i, name in enumerate(names, start=1):
    print(f"{i}. {name}")    # 1. রাহিম, 2. করিম...

# sorted() with key:
students = [
    {"name": "করিম", "score": 90},
    {"name": "রাহিম", "score": 85},
    {"name": "সালমা", "score": 95},
]
ranked = sorted(students, key=lambda s: s["score"], reverse=True)
for s in ranked:
    print(f"{s['name']}: {s['score']}")
